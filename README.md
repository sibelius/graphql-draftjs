# GraphQL Draftjs

GraphQL types for DraftJS

## Install

```sh
yarn install graphql-draftjs
``` 

## Usage
```jsx
import { 
  DraftContentStateType,
  DraftContentStateInput,
  DraftBlock,
  DraftBlockInput,
  DraftEntity,
  DraftEntityInput,
  DraftData,
  DraftDataInput,
  DraftEntityRange,
  DraftEntityRangeInput,
  DraftInlineStyleRange,
  DraftInlineStyleRangeInput,
} from 'draftjs-graphql'
```
